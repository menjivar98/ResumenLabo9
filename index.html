IMPLEMENTAR FIREBASE A NUESTRA APLICACION


Opcion 1

  • Se debe crear una cuenta y acceder

  • Se agrega un proyecto nuevo

  • Se escoge la opcion de Android

  • Se llenan los campos necesarios (El campo de certificado es opcional)

  • Hay que descargar el packet json y pegarlo en la carpeta src en Android Studio (en la vista de "Porject")

  • Se debe de implementar la independencia "implementation 'com.google.firebase:firebase-core:16.0.8'"

Opcion 2

  • Se selecciona la pestaña de Tools -> Firebase

  • Salen opciones de Firebase y se elige la opcion que se desea usar, por ejemplo "Authentication"

  • Seleccionamos "Connect to firebase"

  • Conectamos a una base

  • Al finalizar agregamos Firebase a nuestr app

Firebase y sus Pros

Firebase es una plataforma que ofrece servicios para desarrollo Optimo de apps y apps moviles

Compila mejores apps a traves de :

  • Cloud Firestone

  • Authentication

  • Cloud Messaging

  • Remote Config

  • Y más

Algunas Ventajas:

  • Mejora el rendimiento de la App

  • Minimiza los errores

  • Crea API con facilidad

  • Es multiplataforma

  • Posee Firebase Analytics, que esta enfocado a medir metricas especi­ficas de una aplicacion movil.
